## Usage
**1.** **Download** the project as **ZIP** or **Clone** it via:

**` git clone https://github.com/BakiSejdiu/kscrypto.git `**

**2.** **Install node_modules** in terminal: 

**`npm install `**

**3.** **Start** the project:

**`npm start`**
